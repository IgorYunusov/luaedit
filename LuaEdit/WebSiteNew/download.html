Description:
LuaEdit is power script editor and debugger for the version 5.1 of Lua. It features many tools such as advanced breakpoint management, syntax highlighting, completion proposal list, local and global variable windows. Its interface has been designed to look as close as possible to MS Visual Studio .Net IDE.
